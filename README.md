# gaudi-widgets

gaudi-Übersicht is an [Übersicht](http://tracesof.net/uebersicht/) system information bars. 

The main decision behind developing gaudi was to allow usage of adaptive widgets. If you have been using Übersicht you will know that to customize the location of any widget, you had to dig into the code and position it _most probably with absolute position_. For the case of information bars, I did not find that to be a fluid experience especially when the information is dynamic e.g.,
 
 - Display Wi-Fi information where the network SSID can be long (you can work around this with a maximum width and text overflow .. but really?)
 - System Information, application and opened window names of varying lengths
 ... etc.
